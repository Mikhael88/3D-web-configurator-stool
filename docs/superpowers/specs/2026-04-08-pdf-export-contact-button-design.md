# PDF Export & Contact Button — Design Spec
_Date: 2026-04-08_

## Overview

Replace the "Estimated Price + Finalize Order" bottom section of the ConfigSidebar with two buttons: a PDF export and a contact placeholder. The PDF captures three rendered views of the 3D model plus a text configuration summary, all client-side.

---

## Changes to ConfigSidebar Bottom Area

**Remove:**
- "Estimated Price" label + "$4,290.00" value
- "Finalize Order" button (col-span-4)
- Share icon button (col-span-1)

**Add:** Two equal-width buttons in a `grid-cols-2` layout:

| Button | Style | Action |
|--------|-------|--------|
| Save PDF | Primary — solid `#bcc7de` bg, dark text, PDF icon left | Calls `captureViews()` then generates PDF |
| Contact | Secondary — outlined, `rgba(255,255,255,0.1)` border | No-op for now (placeholder) |

---

## PDF Content (A4 Landscape)

**Page 1:**
- Header: "ATELIER MARITIME" (serif) + model name + ISO date (right-aligned)
- Thin rule separator
- Three images side by side, each labelled below: "Front View", "Side View", "Top View"
- Configuration summary block:
  - Upholstery: name + category (e.g. "Navy Marine Fabric — Fabric")
  - Color: small filled rectangle (upholstery hex) + hex code
  - Structure Finish: "Brushed Steel 316"
  - Armrests: "Yes" / "No"
- Footer: thin rule + "Generated by Atelier Maritime Configurator" + date

## Technical Approach

### Camera capture (`Scene.tsx`)

Module-level function `captureViews()` exposed via a `CaptureHandler` component registered inside the Canvas:

```
Camera positions (all look at [0, 0.7, 0]):
  Front : [0,   1.2, 4.0]
  Side  : [4.0, 1.2, 0  ]
  Top   : [0,   5.5, 0.01]
```

For each position:
1. Set `camera.position`, call `camera.lookAt(0, 0.7, 0)`, `camera.updateMatrixWorld()`
2. Call `gl.render(scene, camera)` directly (bypasses R3F loop — no visual glitch)
3. Capture `gl.domElement.toDataURL('image/jpeg', 0.92)`
4. Restore original camera state

### PDF generation (`ConfigSidebar.tsx`)

Uses `jspdf` (client-side only, no server). The button handler is `async`:
1. Calls `captureViews()` — returns `{ front, side, top }` as base64 JPEG strings
2. Creates `new jsPDF({ orientation: 'landscape', unit: 'mm', format: 'a4' })`
3. Lays out header, images, summary, footer
4. Calls `pdf.save('atelier-maritime-c111.pdf')`

### Exposed interface (module-level, consistent with existing codebase pattern)

```ts
// Scene.tsx (module level)
let captureViewsFn: (() => Promise<{ front: string; side: string; top: string }>) | null = null
export function getCaptureViews() { return captureViewsFn }
```

A `CaptureHandler` component (rendered inside `SceneContent`, returns null) registers and unregisters the function via `useEffect`.

## New Dependency

`jspdf` — client-side PDF generation. No backend required.

Install: `bun add jspdf`
